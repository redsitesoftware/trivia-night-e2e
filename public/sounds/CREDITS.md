# Sound Credits

All sound files in this directory were procedurally generated using Python's
standard `wave` and `math` modules. No third-party audio samples were used.

## Files

### correct.wav
- **Description:** Two-tone ascending chime (E5 → B5) for correct answers
- **Generation:** Procedurally synthesised sine waves with ADSR envelope
- **License:** CC0 1.0 Universal (Public Domain Dedication)
- **Format:** Mono, 22050 Hz, 16-bit PCM

### wrong.wav
- **Description:** Descending buzzer tone (A3 → E3) for wrong answers
- **Generation:** Procedurally synthesised with frequency sweep and ADSR envelope
- **License:** CC0 1.0 Universal (Public Domain Dedication)
- **Format:** Mono, 22050 Hz, 16-bit PCM
